Detailed Description

Associative array template.

Usage:

map<string, int> prg_count = new map<string, int>;
// fill
prg_count.Insert("hello", 10);
prg_count.Insert("world", 20);
prg_count.Insert("!", 1);
Print(prg_count.Get("world")); // prints '20'
